# PR Reviews Analysis

> **Disclaimer**: This code has been generated by AI and may contain errors or limitations. Review it carefully before use.

> **Disclaimer**: This tool is intended for local use only. Do not deploy or expose it to the public internet.

Local tool to analyze PR review activity using GitHub data.

## Requirements

- [Bun](https://bun.sh/) 1.3+
- [GitHub CLI](https://cli.github.com/) installed and authenticated

```bash
gh auth login
gh auth status
```

## Setup

```bash
bun install
cp .env.example .env
```

## Environment Variables (`.env`)

- `PORT`: API server port (default `3000`)
- `DEFAULT_REPOS`: pre-filled repositories (`owner/repo`, comma-separated)
- `DEFAULT_LABEL`: pre-filled label filter
- `DEFAULT_TEAM`: pre-filled team members (comma-separated GitHub handles)
- `CACHE_TTL_HOURS`: cache time-to-live in hours
- `CACHE_DIR`: cache directory path

## Run

```bash
bun run dev
```

- App: [http://localhost:5173](http://localhost:5173)
- API: [http://localhost:3000](http://localhost:3000)

For non-watch mode:

```bash
bun start
```
